Start with the end-to-end tutorial, then use the focused guides as references:

1. [Build a complete plugin](/docs/guides/complete-plugin) — go from scaffold to a validated stereo VST3 bundle.
2. [Parameters](/docs/guides/parameters) — declare stable, host-visible controls.
3. [DSP kernels](/docs/guides/dsp) — process audio and events without violating realtime constraints.
4. [Web UI](/docs/guides/web-ui) — connect an editor through the typed bridge.
5. [State and lifecycle](/docs/guides/state-and-lifecycle) — restore projects and respond to host lifecycle changes.

Each guide focuses on the contract between layers rather than a specific UI framework or DSP algorithm.
